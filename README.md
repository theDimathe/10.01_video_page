# 10.01_video_page

Минимальное окружение для лендинга на HTML, CSS и JS.

## Быстрый старт

1. Установить зависимости:

   ```bash
   npm install
   ```

2. Запустить dev-сервер:

   ```bash
   npm run dev
   ```

Страница будет доступна на `http://localhost:5173`.

## Полезные команды

- `npm run format` — форматирование файлов HTML/CSS/JS/MD.
